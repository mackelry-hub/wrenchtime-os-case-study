# Roadmap

This roadmap describes public-safe product direction. It does not reveal private priorities, sensitive workflows, or unreleased operational details.

## Near-Term

- Continue improving admin dashboard clarity and mobile ergonomics.
- Expand tracking-health checks for attribution and conversion data.
- Improve CPA/year-end export polish.
- Add more explicit review states around AI advisory notes.
- Strengthen test coverage around growth-intelligence calculations and provider boundaries.

## Mid-Term

- Improve read-only connector observability for marketing and banking integrations.
- Add richer internal reporting for quote, schedule, payment, and expense trends.
- Create safer operator workflows for reviewing AI recommendations over time.
- Add stronger audit trails for important admin actions.

## Long-Term

- Build more robust forecasting and planning views from historical operational data.
- Expand internal analytics while preserving privacy and least-privilege integration design.
- Continue separating advisory intelligence from action-taking systems.
- Improve deployment, monitoring, and incident response practices as the platform matures.
